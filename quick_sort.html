⚡ 快速排序法

Quick Sort - 高效分治排序演算法

📌 什麼是快速排序法？

快速排序法（Quick Sort）是一種分治法（Divide and Conquer）排序演算法。它透過選定一個基準值（pivot），將陣列分成兩部分：小於基準值的在左側，大於基準值的在右側，然後對左右子陣列遞迴進行排序，直到整個數列排序完成。

🎯 演算法原理

  1. 選擇基準值（pivot）
  2. 將數列分為左右兩部分：左側小於 pivot，右側大於 pivot
  3. 對左右子陣列遞迴進行快速排序
  4. 合併結果，得到完整排序陣列

💻 代碼示例

C 實現：

#include <stdio.h> void swap(int *a, int *b) { int temp = *a; *a = *b; *b = temp; } int partition(int arr[], int low, int high) { int pivot = arr[high]; int i = low - 1; for(int j = low; j < high; j++) { if(arr[j] <= pivot) { i++; swap(&arr[i], &arr[j]); } } swap(&arr[i+1], &arr[high]); return i+1; } void quick_sort(int arr[], int low, int high) { if(low < high) { int pi = partition(arr, low, high); quick_sort(arr, low, pi - 1); quick_sort(arr, pi + 1, high); } } int main() { int arr[] = {64, 34, 25, 12, 22, 11, 90}; int n = 7; quick_sort(arr, 0, n-1); printf("排序後的陣列: "); for(int i=0; i<n; i++) printf("%d ", arr[i]); return 0; }

📊 複雜度分析

指標複雜度說明
時間複雜度（最差）O(n²)陣列已排序或選 pivot 不佳時
時間複雜度（平均）O(n log n)隨機或平均情況
空間複雜度O(log n)遞迴堆疊深度
穩定性不穩定相等元素相對位置可能改變

✅ 優點與缺點

✓ 優點

  • 效率高，平均時間複雜度 O(n log n)
  • 分治思想易理解，遞迴實現簡單
  • 適合大數據排序
  • 在內存中排序效率高

✗ 缺點

  • 最差情況 O(n²)
  • 不穩定排序
  • 需要遞迴，堆疊深度大可能耗內存

🔍 執行範例過程

初始陣列： [5, 2, 8, 1, 9]

選 pivot = 9，分割後： [5, 2, 8, 1] | 9

遞迴排序左側： pivot = 1 → [1] | [2,5,8]

遞迴排序右側： [2,5,8] pivot=8 → [2,5] | 8

最終排序： [1,2,5,8,9]

💬 重點總結：快速排序法高效且適合大數據，但不穩定，最差情況需要注意。是學習排序演算法中重要的經典範例。
