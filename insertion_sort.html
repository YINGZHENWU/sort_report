📥 插入排序法

Insertion Sort - 一種逐步插入的排序演算法

📌 什麼是插入排序法？

插入排序法（Insertion Sort）是一種簡單且直觀的排序方法，它的概念類似整理撲克牌。每次從未排序部分取出一個元素，插入到已排序部分的正確位置。

🎯 演算法原理

  1. 將第一個元素視為已排序
  2. 取下一個元素，與已排序部分比較
  3. 將元素插入到正確位置
  4. 重複直到所有元素排序完成

💻 代碼示例

C 實現：

#include <stdio.h> void insertion_sort(int arr[], int n) { for(int i = 1; i < n; i++) { int key = arr[i]; int j = i - 1; while(j >= 0 && arr[j] > key) { arr[j + 1] = arr[j]; j--; } arr[j + 1] = key; } } int main() { int arr[] = {12, 11, 13, 5, 6}; int n = 5; insertion_sort(arr, n); printf("排序後的陣列: "); for(int i=0; i<n; i++) printf("%d ", arr[i]); return 0; }

📊 複雜度分析

指標複雜度說明
最差情況O(n²)完全逆序排列
最好情況O(n)已排序數列
平均情況O(n²)隨機排列
空間複雜度O(1)不需額外空間
穩定性穩定相同元素順序不變

✅ 優點與缺點

✓ 優點

  • 簡單易懂
  • 適合小型資料
  • 對幾乎已排序的資料效率高
  • 穩定排序

✗ 缺點

  • 時間複雜度高 O(n²)
  • 不適合大數據

🔍 執行範例

初始： [12, 11, 13, 5, 6]

→ 插入 11 → [11, 12, 13, 5, 6]

→ 插入 13 → [11, 12, 13, 5, 6]

→ 插入 5 → [5, 11, 12, 13, 6]

→ 插入 6 → [5, 6, 11, 12, 13]

💡 重點：插入排序在小資料或接近排序完成時非常有效，但在大數據時效率較差。
