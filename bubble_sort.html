🔄 氣泡排序法

Bubble Sort - 一種簡單但經典的排序演算法

📌 什麼是氣泡排序法？

氣泡排序法（Bubble Sort）是一種簡單的排序演算法，它通過重複走訪數列，逐一比較相鄰元素，若順序不正確則交換，直到整個數列排序完成。名稱來源於較小的元素像氣泡一樣逐步浮到數列頂端。

🎯 演算法原理

  1. 比較相鄰元素，若順序錯誤則交換
  2. 完成一次遍歷後，最大的元素會移到最後
  3. 重複遍歷未排序部分，直到整個數列排序完成

💻 代碼示例

C 實現：

#include <stdio.h> void bubble_sort(int arr[], int n) { for(int i = 0; i < n; i++) { for(int j = 0; j < n-i-1; j++) { if(arr[j] > arr[j+1]) { int temp = arr[j]; arr[j] = arr[j+1]; arr[j+1] = temp; } } } } int main() { int arr[] = {64, 34, 25, 12, 22, 11, 90}; int n = 7; bubble_sort(arr, n); printf("排序後的陣列: "); for(int i=0; i<n; i++) printf("%d ", arr[i]); return 0; }

📊 複雜度分析

指標 複雜度 說明
時間複雜度（最差/平均） O(n²) 逆序排列時，需要進行 n(n-1)/2 次比較
時間複雜度（最好） O(n) 數列已排序，只需一次遍歷
空間複雜度 O(1) 僅需常數額外變數
穩定性 穩定 相等元素的相對位置保持不變

✅ 優點與缺點

✓ 優點

  • 演算法簡單易懂
  • 實現容易，代碼簡潔
  • 不需要額外空間
  • 穩定排序演算法
  • 適合小型數據排序

✗ 缺點

  • 時間複雜度高 O(n²)
  • 大數據集效能差
  • 比較和交換次數多
  • 效率低於快速排序、歸併排序等

🔍 執行範例過程

初始陣列： [5, 2, 8, 1, 9]

第一次遍歷：

  • 比較 5 和 2 → 交換 → [2, 5, 8, 1, 9]
  • 比較 5 和 8 → 無須交換 → [2, 5, 8, 1, 9]
  • 比較 8 和 1 → 交換 → [2, 5, 1, 8, 9]
  • 比較 8 和 9 → 無須交換 → [2, 5, 1, 8, 9]

第二次遍歷： [2, 5, 1, 8, 9]

持續進行直到排序完成： [1, 2, 5, 8, 9]

💬 重點總結：氣泡排序法簡單易懂，但效率低，適合小規模數據或學習排序概念。對大數據集建議使用快速排序、歸併排序或堆排序等更高效演算法。
