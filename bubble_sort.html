🔄 氣泡排序法

Bubble Sort - 一種簡單但經典的排序演算法

📌 什麼是氣泡排序法？

氣泡排序法（Bubble Sort）是一種簡單的排序演算法，它通過重複走訪要排序的數列，比較相鄰的元素，若順序不正確則交換，直到整個數列排序完成。其名稱來自於較小的元素會慢慢「浮」到數列的頂部，就像氣泡浮到水面一樣。

🎯 演算法原理

氣泡排序的工作原理如下：

  1. 比較相鄰元素：從數列的第一個元素開始，逐一比較相鄰的兩個元素
  2. 交換順序：如果前一個元素大於後一個元素，則交換它們
  3. 重複過程：繼續向後比較，每次完整遍歷後，最大的元素會移動到最後位置
  4. 多次遍歷：重複進行2-3步，直到整個數列排序完成

💻 代碼示例

C 實現：

#include <stdio.h> void bubble_sort(int arr[], int n) { for (int i = 0; i < n; i++) { for (int j = 0; j < n - i - 1; j++) { if (arr[j] > arr[j + 1]) { int temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; } } } } int main() { int arr[] = {64, 34, 25, 12, 22, 11, 90}; int n = 7; bubble_sort(arr, n); printf("排序後的陣列: "); for (int i = 0; i < n; i++) printf("%d ", arr[i]); return 0; }

📊 複雜度分析

指標 複雜度 說明
時間複雜度（最差情況） O(n²) 數列逆序排列時，需要進行n(n-1)/2次比較
時間複雜度（最好情況） O(n) 數列已排序，只需遍歷一次
時間複雜度（平均情況） O(n²) 隨機排列的數列
空間複雜度 O(1) 只需要常數個額外的變數
穩定性 穩定 相等元素的相對位置不變
